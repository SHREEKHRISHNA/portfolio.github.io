1.Elements: HTML documents consist of a collection of elements, which are enclosed within opening and closing tags. Elements define the structure and content of the document. Examples of HTML elements include headings (h1,h2,h3), paragraphs images (img), links (a), lists (ul, ol, li), and many more.

2.Attributes: HTML elements can have attributes that provide additional information about the element. Attributes appear within the opening tag and are written as name-value pairs. For example, the img element has attributes such as src (source) and alt (alternative text).

3.Document Structure: HTML documents are structured as a hierarchy of elements. They typically begin with a declaration, followed by an html element that contains head and body elements. The head section includes meta-information about the document, such as the title, character encoding, and links to external resources. The section contains the content of the web page.

4.Semantic Markup: HTML5 introduced several semantic elements that provide meaning to the content, making it more accessible to both humans and machines. Semantic elements include header, footer, nav, article, section, main, aside, and figure, among others.


1.Static HTML: Static HTML output refers to content that is fixed and unchanging. It is directly written into the HTML document and does not change unless the HTML code itself is modified. Static HTML content is suitable for elements like text, images, links, and basic page layout
